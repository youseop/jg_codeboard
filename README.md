> comment model 추가해서 데이터베이스 한번 엎어야 할 수도 있음

기본기능 
- 추천 기능 추가
- 검색, 정렬 기능 - 동은 하는 중 

- 마크다운 기능 적용 -> 적용은 시켰는데 코드 쓸 때 tab 안됨, 코드 줄 안나눠짐 + 인용구 안됨


+ 답변 페이징
+ 소셜로그인

+ 최근 답변, 댓글
